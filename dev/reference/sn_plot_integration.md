# Plot integration assessment metrics

Plot integration assessment metrics

## Usage

``` r
sn_plot_integration(x, aggregate = FALSE)
```

## Arguments

- x:

  Result from
  [`sn_assess_integration()`](https://songqi.org/shennong/dev/reference/sn_assess_integration.md)
  or its summary table.

- aggregate:

  Include aggregate rows.

## Value

An integration score plot.
