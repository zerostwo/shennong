# Plot a clustering resolution sweep

Plot a clustering resolution sweep

## Usage

``` r
sn_plot_resolution_sweep(x, metrics = NULL)
```

## Arguments

- x:

  Result from
  [`sn_sweep_cluster_resolution()`](https://songqi.org/shennong/dev/reference/sn_sweep_cluster_resolution.md)
  or its summary table.

- metrics:

  Numeric metric columns to display.

## Value

A resolution-quality plot.
