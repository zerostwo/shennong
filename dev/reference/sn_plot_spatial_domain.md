# Plot spatial-domain assignments

Plot spatial-domain assignments

## Usage

``` r
sn_plot_spatial_domain(x, name = NULL, point_size = 1.5)
```

## Arguments

- x:

  A Seurat object or spatial-domain result.

- name:

  Stored result name.

- point_size:

  Point size.

## Value

A `ggplot` object.
