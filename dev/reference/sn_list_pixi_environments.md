# List bundled pixi environment configs

List bundled pixi environment configs

## Usage

``` r
sn_list_pixi_environments()
```

## Value

A character vector of environment names with pixi manifests bundled
under `inst/pixi/`.

## Examples

``` r
sn_list_pixi_environments()
#>  [1] "cell2location" "cellphonedb"   "infercnvpy"    "mmochi"       
#>  [5] "scarches"      "scvi"          "spatialdata"   "squidpy"      
#>  [9] "stlearn"       "tangram"      
```
