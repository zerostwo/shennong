# List bundled pixi environment configs

List bundled pixi environment configs

## Usage

``` r
sn_list_pixi_environments()
```

## Value

A character vector of environment names with pixi manifests bundled
under `inst/pixi/`.

## Examples

``` r
sn_list_pixi_environments()
#> [1] "cell2location" "cellphonedb"   "infercnvpy"    "scarches"     
#> [5] "scvi"          "spatialdata"   "squidpy"       "stlearn"      
#> [9] "tangram"      
```
