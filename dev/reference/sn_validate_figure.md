# Validate a Shennong figure before export

Validate a Shennong figure before export

## Usage

``` r
sn_validate_figure(plot, profile = NULL, error = FALSE)
```

## Arguments

- plot:

  A figure with or without an attached Shennong specification.

- profile:

  Optional output profile override.

- error:

  Stop when validation finds an error-level problem.

## Value

A structured validation report.
