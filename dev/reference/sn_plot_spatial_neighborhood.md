# Plot spatial-neighborhood enrichment or co-occurrence

Plot spatial-neighborhood enrichment or co-occurrence

## Usage

``` r
sn_plot_spatial_neighborhood(
  x,
  name = NULL,
  type = c("enrichment", "cooccurrence")
)
```

## Arguments

- x:

  A Seurat object or spatial-neighborhood result.

- name:

  Stored result name.

- type:

  Enrichment or co-occurrence.

## Value

A `ggplot` object.
