# Plot PCA elbow diagnostics

Plot PCA elbow diagnostics

## Usage

``` r
sn_plot_elbow(object, reduction = "pca", ndims = 50L)
```

## Arguments

- object:

  A Seurat object.

- reduction:

  Reduction containing standard deviations.

- ndims:

  Number of dimensions.

## Value

An elbow plot with figure metadata.
