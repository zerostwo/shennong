# Plot highly variable feature diagnostics

Plot highly variable feature diagnostics

## Usage

``` r
sn_plot_hvg(object, assay = NULL, label_n = 10L)
```

## Arguments

- object:

  A Seurat object with variable features.

- assay:

  Assay name.

- label_n:

  Number of top variable features labeled.

## Value

A Seurat variable-feature plot with figure metadata.
