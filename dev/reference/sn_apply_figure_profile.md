# Apply a generic publication profile to a plot

Apply a generic publication profile to a plot

## Usage

``` r
sn_apply_figure_profile(plot, profile, ...)
```

## Arguments

- plot:

  A ggplot or patchwork object.

- profile:

  Figure profile name.

- ...:

  Explicit figure-spec overrides.

## Value

The original plot with profile styling and an attached specification.
