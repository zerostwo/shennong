# Plot annotation marker evidence

Plot annotation marker evidence

## Usage

``` r
sn_plot_annotation_markers(x, store_name = "annotation", top_n = 5)
```

## Arguments

- x:

  A Seurat object or annotation result.

- store_name:

  Stored annotation name.

- top_n:

  Number of candidate labels shown per cluster.

## Value

A `ggplot` object.

## Examples

``` r
if (FALSE) sn_plot_annotation_markers(object) # \dontrun{}
```
