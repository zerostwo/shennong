# Plot CellRank fate probabilities

Plot CellRank fate probabilities

## Usage

``` r
sn_plot_fate(x, name = NULL, states = NULL, point_size = 0.7)
```

## Arguments

- x:

  A Seurat object or fate result.

- name:

  Stored result name.

- states:

  Optional terminal states to retain.

- point_size:

  Point size.

## Value

A faceted `ggplot` object.
