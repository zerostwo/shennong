# Store local CIBERSORTx credentials for container execution

Store local CIBERSORTx credentials for container execution

## Usage

``` r
sn_set_cibersortx_credentials(email, token)
```

## Arguments

- email:

  Email registered with CIBERSORTx.

- token:

  Access token issued by CIBERSORTx.

## Value

Invisibly returns `TRUE`.
