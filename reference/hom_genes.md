# Human-Mouse Homologous Genes Table

Human and mouse homolog mappings derived from the MGI homology report.

## Usage

``` r
hom_genes
```

## Format

A data frame with 5 columns:

- `DB.Class.Key`:

  MGI homology-group identifier.

- `human`:

  Approved human gene symbol.

- `HGNC.ID`:

  HGNC identifier for the human gene.

- `mouse`:

  Approved mouse gene symbol.

- `Mouse.MGI.ID`:

  MGI identifier for the mouse gene.

## Source

MGI homology report:
<https://www.informatics.jax.org/downloads/reports/HOM_MouseHumanSequence.rpt>
